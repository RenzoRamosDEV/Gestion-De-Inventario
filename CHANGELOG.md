# Histórico de Cambios

## [1.0.0] - 2026-04-30

### Añadido
- Sistema completo de gestión de inventario
- Backend con Spring Boot 3.2 y Java 17
- Frontend con React 18 y TypeScript
- Base de datos MySQL con esquemas optimizados
- API REST con documentación Swagger/OpenAPI
- Operaciones CRUD para productos
- Gestión de stock con validaciones
- Dashboard con estadísticas en tiempo real
- Sistema de paginación y filtros
- Soft delete para productos
- Auto-restauración de productos al aumentar stock
- Manejo global de errores
- Endpoints de monitoreo con Spring Boot Actuator
- Configuración Docker y Docker Compose
- Pipeline CI/CD con GitHub Actions
- Scripts de automatización para desarrollo
- Componentes UI reutilizables en React
- Sistema de notificaciones con toasts
- Validaciones tanto en backend como frontend

### Características Técnicas
- Arquitectura limpia con separación de capas
- Patrón Repository para acceso a datos
- Mappers entre DTOs y entidades
- Validaciones personalizadas
- Logs estructurados
- Health checks personalizados
- Configuración por perfiles (dev/prod)
- Variables de entorno
- Estilos con Tailwind CSS
- Componentes TypeScript tipados
- Context API para estado global

### Infraestructura
- Contenedorización completa
- Configuración de nginx para producción
- Scripts de build y deployment
- Monitoreo y métricas
- Documentación técnica completa
- Guías de contribución
